# ATAQ_QGIS_PlugIn

QGIS plugin for ATAQ AERMOD interface.

ReadMe to be further developed/

In a nutshell: The plugin offers 3 functions. The fist creates 3 layes. Polygons for area sources, points for point sources and lines for line sources.

You can then add features to these layers which will pop-up a template to assist with providing the  information to complete the attribute table that requires the information needed by AERMOD.  

Finally, the 3 layers can be exported and saved as csv's that can be read by  ATAQ_AERMOD.
